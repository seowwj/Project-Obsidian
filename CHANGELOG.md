# Project Obsidian Release Notes

## [0.1.0] - 2026-01-13

Initial release

### Supported Features
- General chat
- Video / Audio summary
- Video / Audio question and answer
- Chat history

### Known Issues / Limitations
- Not able to chat with multiple files at once
- Not able to select / choose own models
- Not able to reference memory / history across different chat sessions
- Requires models to already be downloaded using separate script
- Runs fully on CPU (no GPU)
- No hard limit on uploaded file size
- No dynamic loading of models
